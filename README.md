## ğŸƒâ€â™‚ï¸ Getting Started

### Prerequisites

*   Ruby (with `bundler`)
*   Node.js

### Local Development

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/tugcanonbas/uxdm_vcux-hackathon-tugcanonbas.git
    cd uxdm_vcux-hackathon-tugcanonbas
    ```

2.  **Install dependencies:**
    ```bash
    bundle install
    ```

3.  **Run the server:**
    ```bash
    bundle exec jekyll serve
    ```
    The site will be available at `http://localhost:4000`.

### Using Docker

Alternatively, you can run the site inside a Docker container:

```bash
docker-compose up
```
